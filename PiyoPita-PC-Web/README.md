# ぴよぴた PC版（HTML/CSS/JavaScript）

PC向けの「ぴよぴた」歯科医師・医院用管理画面のプロトタイプです。  
HTML・CSS・JavaScriptのみで構成され、ビルド不要でブラウザでそのまま開けます。

## 構成

- `index.html` - エントリーページ
- `css/styles.css` - スタイル（ダッシュボード・カード・テーブル等）
- `js/app.js` - 認証・ルーティング・画面描画・状態管理

## 画面

- **ログイン** - 医院ID・歯科医師ID・パスワード（sessionStorage で認証状態を保持）
- **ホーム** - お知らせタイムライン・連続未装着アラート（5日以上/3〜4日/2日）
- **患者検索** - 検索ワード・使用装置フィルタ・一覧テーブル
- **患者詳細** - 装置設定・治療進捗・装着達成率（CSS棒グラフ）・装着カレンダー・日付タップで写真モーダル
- **メッセージ** - 患者リスト・チャット
- **設定** - 医院情報 / アカウント（ログイン・通知） / スタッフ管理タブ

ヘッダーに「歯科医院情報QR」ボタンがあり、クリックでQRモーダルを表示します。

## 起動方法

1. フォルダ内の `index.html` をブラウザで開く  
2. またはローカルサーバーで配信（例: `npx serve .`）して `http://localhost:3000` を開く

未ログイン時は `#login` にリダイレクトされます。ログイン画面で任意の値を入力して「ログイン」を押すとダッシュボードに遷移します。  
ルーティングはハッシュ（`#/`, `#/patients`, `#/patients/1`, `#/messages`, `#/settings`）で行っています。
