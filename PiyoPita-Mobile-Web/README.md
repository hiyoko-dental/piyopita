# ぴよぴた モバイル版（HTML/CSS/JavaScript）

スマートフォン向けの「ぴよぴた」患者用Webアプリのプロトタイプです。  
HTML・CSS・JavaScriptのみで構成され、ビルド不要でブラウザでそのまま開けます。

## 構成

- `index.html` - エントリーページ
- `css/styles.css` - スタイル（ぴよぴたカラー・レイアウト）
- `js/app.js` - ルーティング・画面描画・状態管理

## 画面

- **トップ** - ロゴ、歯科医師との連携 / データ引継ぎ / お問い合わせ
- **利用規約** - 規約同意後「歯科医師と連携開始」
- **QR読み取り** - 歯科医院QRスキャン（シミュレート）
- **患者情報登録** - 氏名・ニックネーム・生年月日
- **カレンダー** - 装着記録の月表示・日付タップで詳細モーダル
- **記録する** - 装置QRスキャン / シール剥がれ
- **記録確認** - 評価・日時・決定/再撮影
- **ぴよぴたゲーム** - 連続記録・総記録・バッジ
- **メニュー** - アカウント・装置について・ヘルプ・お知らせ・ログアウト
- **使用装置について** - バイオネータ説明・注意点

## 起動方法

1. フォルダ内の `index.html` をブラウザで開く  
2. またはローカルサーバーで配信（例: `npx serve .`）して `http://localhost:3000` を開く

ルーティングはハッシュ（`#/calendar` など）で行っています。

## フォント

Google Fonts の「M PLUS Rounded 1c」を読み込んでいます（CDN）。
